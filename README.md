# Instancia de Evaluacion 2
Programación - 1º año ITSC Desarrollo de Software

Consignas de evaluación: 
Desarrollar una aplicación en lenguaje C# con el entorno de desarrollo de software Visual Studio 2019. La misma debe reflejar un hecho de la realidad y cuyo modelo de solución implique: 
  - Programación en Capas (front end y back end). 
  - Uso de al menos una clase diferente del formulario inicial. 
  - Persistencia de los datos. 
  
El programa desarrollado deberá ser publicado en GitHub y cuyo enlace deberá se comunicado al momento de la entrega.
